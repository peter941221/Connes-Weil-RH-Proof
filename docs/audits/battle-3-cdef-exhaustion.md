## Target Definition

For fixed finite `S`, compact support interval `I`, cutoff `lambda`, and graph
order `J`, define a finite endpoint-strip index set:

```text
R_(S,I,lambda,J).
```

Each element has the form:

```text
alpha=(r,s,a,X_0,X_1,b,T_a),
```

with:

```text
r+s <= J,
supp(b) subset E_(lambda,a),
X_0,X_1 fixed graph-bounded factors,
T_a a bounded shift/dilation factor.
```

Then define:

```text
Cdef_(S,I,lambda,J)(g)
  :=
sum_(alpha in R_(S,I,lambda,J))
  || theta(D^r g) X_0 M_b T_a X_1 theta(D^s g)^* ||_1

+

sum_(beta in Q R_(S,I,lambda,J))
  |BoundaryStripTrace_beta(g)|.
```

Every `BoundaryStripTrace_beta(g)` must contain at least one endpoint-strip
factor before boundary evaluation. Otherwise `Q` may create a new no-strip
rank or pole jet, which would not be killed by the current ledger.

## Required Estimate

The endpoint-strip trace ideal theorem must prove:

```text
Cdef_(S,I,lambda,J)(g)
  <=
C'_(S,I,J)(g) Cdef_graph_(S,I,lambda,J')(g).
```

Then the fixed-test exhaustion theorem must prove:

```text
Cdef_graph_(S_A,I,lambda,J')(g) -> 0
```

for fixed:

```text
g,
I,
S_A.
```

Since `C'_(S_A,I,J)(g)` is fixed in that limit, the trace-norm `Cdef` also
vanishes.

## Subclaim 1. Endpoint-Strip Normal Form

Required theorem:

```text
EndpointStripNormalForm(S,I,lambda,J):
  every fixed-S projection-order defect generated by
  [P,M_S], [P_hat,M_S], [P,M_S^*], or [P_hat,M_S^*]
  expands into finitely many endpoint-strip normal-form terms.
```

The target normal form is:

```text
theta(D^r g) X_0 M_b T_a X_1 theta(D^s g)^*.
```

This theorem must name the finite index set `R_(S,I,lambda,J)`. A role-only
definition of `Cdef` does not pass.

## Subclaim 2. Trace-Norm Control

Required theorem:

```text
EndpointStripTraceNormBound(S,I,lambda,J):
  every endpoint-strip normal-form term is trace-class and satisfies the
  displayed trace-norm bound.
```

The proof must factor each term through the endpoint strip:

```text
L2(R) --B_s--> L2(E_(lambda,a)) --A_r--> L2(R),
```

then use:

```text
||A_r B_s||_1 <= ||A_r||_2 ||B_s||_2.
```

The finite-S Euler coefficient budget may enter the constant
`C_(S,I,J)(g)`, but it cannot depend on `lambda` in a way that blocks the
fixed-test limit.

## Subclaim 3. Q Stability

Required theorem:

```text
QEndpointStripStability(S,I,lambda,J):
  applying the Connes--Consani Q trace-remainder extraction to an endpoint-strip
  defect keeps at least one endpoint-strip factor before each boundary
  evaluation.
```

This prevents `Q` from creating a new no-strip boundary jet. Pure no-strip
rank and Tate pole terms must come only from the no-defect source calculation
and must already be accounted for by Battles 1 and 2.

## Subclaim 4. Graph Comparison

Required theorem:

```text
CdefGraphComparison(S,I,lambda,J,J'):
  Cdef_(S,I,lambda,J)(g)
    <=
  C'_(S,I,J)(g) Cdef_graph_(S,I,lambda,J')(g).
```

This theorem connects the trace-norm `Cdef` used in Theorem 2 to the
graph/prolate defect used by the exhaustion input.

It must state which graph order `J'` is needed after `Q` raises the order. It
must also state which seminorms of the fixed test `g` enter the constant.

## Subclaim 5. Fixed-Test Exhaustion

Required theorem:

```text
FixedTestCdefExhaustion(S_A,I,g,J'):
  Cdef_graph_(S_A,I,lambda,J')(g) -> 0
  as lambda -> infinity,
  with g, I, and S_A fixed.
```

The order of limits matters:

```text
choose g
choose I containing supp(g)
choose S_A from supp(g^* * g)
then send lambda -> infinity
```

The route fails if `S` grows with `lambda` during this limit or if the graph
order grows without a fixed bound.

## Rejection Tests

Battle 3 fails if any of these occur:

| rejection test | why it kills the route |
|---|---|
| `Cdef` is defined only by its role as "the error" | Theorem 2 has no normed quantity to estimate |
| `R_(S,I,lambda,J)` is not finite for fixed parameters | the endpoint-strip sum may not define a controlled trace norm |
| `Q` can remove the last endpoint-strip factor before boundary evaluation | new no-strip rank or pole jets appear |
| `C'_(S,I,J)(g)` depends on `lambda` in the fixed-test limit | graph/prolate exhaustion no longer implies trace-norm exhaustion |
| `Cdef_graph -> 0` is assumed without a fixed-test theorem | the final limit imports the core unresolved spectral leakage problem |
| `S_A` is allowed to grow with `lambda` | the finite-prime stabilization argument breaks |
