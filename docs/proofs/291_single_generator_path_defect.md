# Proof 291: Single-generator path defect reduction

Date: 2026-07-16

Status: exact algebraic refinement of Proof 290.  Every detector defect along
the finite renewal path is generated by one fixed physical intertwinement
defect and its left companion.  For a diagonal root detector, the two are
adjoints.  No new detector commutator appears as the horizon grows.

The remaining theorem is a uniform root-completed estimate for the stopped
path transform of that fixed generator.  This proof does not establish the
estimate, Gate 3U, the finite-`S` sign, the arithmetic same-object identity,
Burnol's identity, or RH.

## 2. Base intertwinement defects

Retain

```text
K=E A iota_B,
Gamma=K*K,
Delta=I-Gamma,
W_B=iota_B* W iota_B.                                (BB.2)
```

Define the right and left defects

```text
d_W= W K-K W_B,                                      (BB.3)

d_W^left=K* W-W_B K*.                                (BB.4)
```

For a cross-root detector, `(BB.3)` and `(BB.4)` are independent orientations
of the same completed crossing.  If `W=W*`, then

```text
d_W^left=d_W*.                                       (BB.5)
```

The scalar fixed mode has `d_I=0` exactly.

## 3. The defect of Delta has no new owner

Since `Delta=I-K*K`, direct multiplication gives

```text
[W_B,Delta]
 =W_B Delta-Delta W_B

 =Gamma W_B-W_B Gamma

 =d_W^left K-K* d_W.                                 (BB.6)
```

Thus the only band commutator introduced by the renewal is already generated
by the two base intertwinement defects.  Equation `(BB.6)` preserves the
ordered convention; neither term is estimated separately.

For `k>=1`, the standard power rule gives

```text
[W_B,Delta^k]
 =sum_(j=0)^(k-1)
   Delta^j [W_B,Delta] Delta^(k-1-j).                 (BB.7)
```

Substituting `(BB.6)` into `(BB.7)` introduces no prime, relative mode, or
new source boundary.

## 4. Every emission block is generated by d_W

Proof 290's right path has emission blocks `K Delta^k`.  The corresponding
block of

```text
E_n:=O_nR_n-R_nW_B                                   (BB.8)
```

is

```text
E_(n,k)
 =W K Delta^k-K Delta^k W_B.                         (BB.9)
```

Add and subtract `K W_B Delta^k`:

```text
E_(n,k)
 =d_W Delta^k+K[W_B,Delta^k].                        (BB.10)
```

Using `(BB.6)--(BB.7)`,

```text
E_(n,k)
 =d_W Delta^k

  +K sum_(j=0)^(k-1)
     Delta^j(d_W^left K-K*d_W)Delta^(k-1-j).          (BB.11)
```

For `k=0`, the sum is empty and `E_(n,0)=d_W`.

The final survivor block is

```text
E_(n,tail)
 =[W_B,Delta^(n+1)]                                   (BB.12)
```

and receives the same expansion `(BB.7)` with `k=n+1`.  Equations
`(BB.11)--(BB.12)` prove that the whole path defect is a linear transform

```text
E_n=T_n(d_W,d_W^left)                                (BB.13)
```

with coefficients made only from `K` and `Delta`.

## 5. Physical source expansion of the generator

Use `B=iota_B iota_B*`, `I-B=C+R`, and commutation of the whole-line
convolutions `W` and `A`.  Starting from `(BB.3)`,

```text
d_W
 =W E A iota_B-E A iota_B W_B

 =[W,E]A iota_B+E A(I-B)W iota_B.                    (BB.14)
```

The two pieces of `I-B=C+R` are detector crossings:

```text
C W iota_B=C[W,E]iota_B,

R W iota_B=-R[W,R]iota_B.                            (BB.15)
```

Hence

```text
d_W
 =[W,E]A iota_B
  +E A C[W,E]iota_B
  -E A R[W,R]iota_B.                                 (BB.16)
```

These are the outer orientation, the returned outer crossing, and the
complete Sonin crossing.  Inserting

```text
R=E E_hat E-K_prol                                   (BB.17)
```

recovers the second-support and prolate pieces, but they must remain in the
single generator `(BB.16)`.  Proof 258 forbids branchwise norms.

For compact roots, `[W,E]` has displacement support `2B_root`; `[W,R]` uses
the same completed CC20 outer/scattering/prolate readback from Proof 261.
Thus the horizon has introduced no wider detector support.

## 6. Correct successor to Proof 290

Proof 290 gives

```text
sum_(k=0)^n N_W Delta^k
 =F_n*E_n+L_n*(I-P_n)E_n.                             (BB.18)
```

Substitute `(BB.13)`:

```text
sum_(k=0)^n N_W Delta^k

 =F_n*T_n(d_W,d_W^left)
  +L_n*(I-P_n)T_n(d_W,d_W^left).                     (BB.19)
```

For diagonal roots, `(BB.5)` reduces `(BB.19)` to one physical generator and
its adjoint.  Proof 263's complex polarization then recovers cross roots
without increasing support.

The active source theorem is

```text
sup_(finite S,n)
 |Tr_root(
    F_n*T_n(d_W,d_W^left)
    +L_n*(I-P_n)T_n(d_W,d_W^left))|

 <=C(1+B_root)^d
   norm(eta)_(H^r)norm(xi)_(H^r).                    (BB.20)
```

The allowed proof order is

```text
construct d_W as the complete physical crossing (BB.16)
  -> form the whole path transform T_n
  -> project only the complete transform with I-P_n
  -> pair with F_n and L_n
  -> apply compact support to the scalar
  -> one absolute value.                               (BB.21)
```

Forbidden orders are

```text
norm each term in the j-sum;
norm the three branches of d_W;
bound norm(L_n) or norm(Z_n);
expand Delta into primes before support;
identify P_n with Burnol's fixed projection.           (BB.22)
```
