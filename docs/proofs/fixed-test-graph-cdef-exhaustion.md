# Fixed-Test Graph Cdef Exhaustion

Status: proof package for the graph/prolate exhaustion input used by Battle 3.

This file proves the fixed-test statement:

```text
Cdef_graph_(S_A,I,lambda,J')(g) -> 0
```

with `g`, `I`, `S_A`, and `J'` fixed before `lambda -> infinity`.

It does not assert spectral convergence of finite operators to zeta zeros.
It uses only fixed-test cutoff exhaustion in support/Fourier-support graph
norms.

## Target

Given:

```text
g in C_c^\infty(R_+^*),
I containing supp(g),
supp(g^* * g) subset exp([-A,A]),
S_A={infinity} union {p : log p <= A},
```

prove:

```text
FixedTestGraphCdefExhaustion(S_A,I,g,J'):
  Cdef_graph_(S_A,I,lambda,J')(g) -> 0
  as lambda -> infinity.
```

## Definitions

Work in logarithmic coordinates. The multiplicative window

```text
[lambda^(-1),lambda]
```

becomes:

```text
[-L,L],  L=log lambda.
```

Let:

```text
P_L
```

be multiplication by the support window `[-L,L]`, and let:

```text
P_hat_L
```

be the Fourier-side window cutoff transported to the same graph model.

For fixed finite `S_A` and fixed graph order `J'`, `Cdef_graph` is a finite
sum of graph seminorms of tails created by:

```text
(1-P_L),
(1-P_hat_L),
[P_L,M_S],
[P_hat_L,M_S],
[P_L,M_S^*],
[P_hat_L,M_S^*],
```

applied to finitely many fixed graph derivatives of `theta_S(g)` and its
adjoint.

## Lemma 1. Fixed Support Tail Vanishes

Statement:

```text
SupportTailExhaustion(g,J'):
  ||(1-P_L) D^k theta_S(g)||_2 -> 0
```

for every fixed graph derivative order `k <= J'`.

Proof.

The test `g` is compactly supported. After moving to logarithmic coordinates,
all fixed graph derivatives of the source test live in a fixed compact
interval. For all sufficiently large `L`, that compact interval lies inside
`[-L,L]`. Hence:

```text
(1-P_L) D^k theta_S(g)=0
```

for all large `L`.

The same argument applies after fixed finite-S multiplication by `M_S` or
`M_S^*`, since `S_A` is fixed and those multipliers are bounded on the fixed
graph domain.

## Lemma 2. Fixed Fourier Tail Vanishes

Statement:

```text
FourierTailExhaustion(g,J'):
  ||(1-P_hat_L) D^k theta_S(g)||_2 -> 0
```

for every fixed `k <= J'`.

Proof.

A compactly supported smooth test has a Schwartz Fourier/Mellin transform in
the logarithmic coordinate. Each fixed graph derivative multiplies that
transform by a fixed polynomial. Therefore the Fourier-side graph tail:

```text
||(1-P_hat_L) D^k theta_S(g)||_2
```

is the `L^2` tail of a Schwartz function outside `[-L,L]`. It tends to zero as
`L -> infinity`.

Since `S_A` is fixed, multiplying by `M_S`, `M_S^*`, or finitely many fixed
Euler graph factors preserves Schwartz-tail decay up to a fixed polynomial
weight.

## Lemma 3. Endpoint-Strip Commutator Tails Vanish

Statement:

```text
EndpointStripCommutatorExhaustion(S_A,I,g,J'):
  every graph seminorm generated by

    [P_L,M_S],
    [P_hat_L,M_S],
    [P_L,M_S^*],
    [P_hat_L,M_S^*]

  tends to zero on the fixed test g.
```

Proof.

For fixed finite `S_A`, expand each Euler multiplier into a finite or
absolutely summable fixed coefficient family of shifts and bounded graph
factors. A commutator with a support or Fourier cutoff has the form:

```text
[P_L,T_a]f
```

or a bounded fixed graph factor times such a term. The support of
`[P_L,T_a]f` lies in a shifted endpoint strip near `+L` or `-L`.

For support-side tails, Lemma 1 kills the term for all large `L` when `f` is a
fixed compactly supported graph derivative.

For Fourier-side tails, Lemma 2 gives the same conclusion in norm: the strip
near the Fourier boundary samples only the Schwartz tail of a fixed
Fourier-side graph derivative.

Summing the fixed finite-S coefficient budget preserves convergence to zero.

## Lemma 4. Graph Cdef Exhaustion

Statement:

```text
Cdef_graph_(S_A,I,lambda,J')(g) -> 0.
```

Proof.

By definition, `Cdef_graph` is a finite sum of graph seminorms of the fixed
families controlled in Lemmas 1 through 3. For each summand:

```text
tail_summand(lambda) -> 0.
```

The number of summands and the fixed finite-S coefficient budget do not grow
with `lambda`, because `S_A` and `J'` are fixed. Therefore the whole finite sum
tends to zero:

```text
Cdef_graph_(S_A,I,lambda,J')(g) -> 0.
```

## Consequence For Battle 3

Battle 3 already proves:

```text
Cdef_(S_A,I,lambda,J)(g)
  <=
C'_(S_A,I,J)(g) Cdef_graph_(S_A,I,lambda,J')(g).
```

The constant is fixed in the fixed-test limit. Combining this comparison with
Lemma 4 gives:

```text
Cdef_(S_A,I,lambda,J)(g) -> 0.
```

Thus Battle 3 no longer needs an opaque graph/prolate exhaustion assumption.
It uses the fixed-test graph cutoff exhaustion proved here.
