# Proof 293: Observable two-boundary path compression

Date: 2026-07-16

Status: exact finite-horizon refinement of Proofs 290--292.  Every emission
slot may be compressed from `W` to `E W E` inside the ordered biorthogonal
path response.  The part of Proof 292's generator which escapes into `C`
is orthogonal to both path frames and contributes exactly zero.  The remaining
observable generator is one completed outer-return minus Sonin boundary pair.

This proof does not establish the uniform stopped-path estimate, Gate 3U, the
finite-`S` sign, the arithmetic same-object identity, Burnol's identity, or RH.

## 1. Result

Use the Proof 290 path maps

```text
R_n x=(Kx,K Delta x,...,K Delta^n x,Delta^(n+1)x),
L_n x=(Kx,Kx,...,Kx,x),                              (BD.1)
```

and its observable

```text
O_n=diag(W,...,W,W_B).                               (BD.2)
```

Since `K=E K` and `K*=K*E`, define the compressed observable

```text
O_n^E=diag(E W E,...,E W E,W_B).                    (BD.3)
```

Then the ordered response is unchanged as an operator on the band space:

```text
L_n* O_n R_n-W_B
 =L_n* O_n^E R_n-W_B.                                (BD.4)
```

No trace cycle is used in `(BD.4)`.  Both sides retain Proof 264's Gram
ordering at every finite horizon.

Let Proof 292's two-boundary defect be

```text
d_W=WK-KW_B.
```

Its observable and escaping parts are

```text
g_W=E d_W,
q_W=C d_W=C W K,
d_W=g_W+q_W.                                         (BD.5)
```

The entire path generated by `q_W` is invisible to both path frames.  Thus the
active path theorem needs only `g_W` and its left companion.

## 2. Exact path invisibility of the escaping output

Put

```text
Q_(W,n)x
 =(q_W x,q_W Delta x,...,q_W Delta^n x,0).           (BD.6)
```

The range of every emission block in `(BD.6)` lies in `C`, while the range of
`K` lies in `E`.  Hence

```text
K* q_W=0,
R_n* Q_(W,n)=0,
L_n* Q_(W,n)=0.                                      (BD.7)
```

For the canonical right-path projection

```text
P_n=R_n(R_n*R_n)^(-1)R_n*,                           (BD.8)
```

equation `(BD.7)` also gives

```text
P_n Q_(W,n)=0.                                       (BD.9)
```

Therefore both terms in Proof 290's canonical/off-range split annihilate the
escape path:

```text
F_n* Q_(W,n)=0,
L_n*(I-P_n)Q_(W,n)=0.                                (BD.10)
```

This is stronger than saying that the final scalar trace of the escape is
zero.  It is an exact operator-level deletion before the trace and before
`n->infinity`.

## 3. Observable generator and its left companion

Define

```text
g_W=E d_W,
g_W^left=d_W^left E.                                 (BD.11)
```

Because `K=E K` and `K*=K*E`, Proof 291's band commutator becomes

```text
[W_B,Delta]
 =g_W^left K-K* g_W.                                 (BD.12)
```

No escaping factor remains in `(BD.12)`.  The visible emission block is

```text
E_(n,k)^vis
 =g_W Delta^k
  +K sum_(j=0)^(k-1)
     Delta^j(g_W^left K-K*g_W)Delta^(k-1-j).          (BD.13)
```

The tail block is the corresponding power commutator at `k=n+1`.  Stacking
these blocks gives

```text
E_n^vis=O_n^E R_n-R_nW_B.                            (BD.14)
```

The original defect is exactly

```text
O_nR_n-R_nW_B=E_n^vis+Q_(W,n).                       (BD.15)
```

Equations `(BD.10)` and `(BD.15)` prove the refined ordered path identity

```text
sum_(k=0)^n N_W Delta^k
 =F_n*E_n^vis+L_n*(I-P_n)E_n^vis.                    (BD.16)
```

For `W=W*`, `g_W^left=g_W*`.  Cross roots are recovered by complex
polarization after the diagonal estimate, without support enlargement.

## 4. Physical two-boundary readback

Proof 292 gives

```text
d_W
 =[W,E]A iota_B
  -E A R[W,R]iota_B.                                 (BD.17)
```

Use

```text
E[W,E]=-E W C.                                       (BD.18)
```

The observable generator is therefore

```text
g_W
 =-E W C A iota_B
  -E A R[W,R]iota_B.                                 (BD.19)
```

Its left companion is

```text
g_W^left
 =-iota_B* A* C W E
  +iota_B*[W,R]R A*E.                                (BD.20)
```

The first term in `(BD.19)` is a completed outer return: `A` first leaks from
the source band into `C`, and the compact-root detector returns across the
outer boundary into `E`.  The second term is the complete Sonin boundary.
The one-way escape `C W K` has disappeared from the ordered response.

Multiplying `(BD.20)` by `K` recovers Proof 285's support-first numerator.
Indeed `iota_B*R=0`, so

```text
g_W^left K
 =-iota_B*A*C W K+iota_B*W R A*K
 =N_W.                                               (BD.20a)
```

Thus the observable path compression preserves the already accepted
outer-minus-Sonin scalar owner; it does not introduce a competing endpoint
formula.

For a root cross-correlation with

```text
supp(F) subset [-2B_root,2B_root],
```

the kernel of `E W C` is confined to opposite half-lines with
`|x-y|<=2B_root`.  This is the precise compact-support stopping point for the
outer-return branch.

The Sonin term still uses the complete signed decomposition

```text
[W,R]
 =[W,E]E_hat E
  +E[W,E_hat]E
  +E E_hat[W,E]
  -[W,K_prol].                                       (BD.21)
```

Never estimate the terms in `(BD.21)` separately.

## 5. Why this does not erase the trace anomaly

The forbidden polar step would replace

```text
K* W K Gamma^(-1)
```

by a cyclically reordered polar expression at the infinite endpoint.  Proof
264 gives a unilateral-shift counterexample to that move.

Proof 293 performs a different operation:

```text
finite horizon first
  -> retain L_n* (left) and R_n (right) ordering
  -> compress W only between ranges already contained in E
  -> delete a vector orthogonal to both path frames
  -> take no infinite-dimensional trace cycle.        (BD.22)
```

Consequently `(BD.4)` preserves the outer arithmetic anomaly.  It only removes
an emission component which the ordered owner never observes.

## 6. Correct successor theorem

```text

    F_n*E_n^vis
    +L_n*(I-P_n)E_n^vis)|

   norm(eta)_(H^r)norm(xi)_(H^r),                    (BD.23)
```

where `E_n^vis` is generated only by `(BD.19)--(BD.20)`.

The legal order is

```text
compress every emission detector to E W E
  -> build the full visible path defect
  -> retain the signed outer-return/Sonin pair
  -> apply I-P_n after assembly
  -> use compact root support on the scalar
  -> take one absolute value
  -> pass to the horizon limit.                       (BD.24)
```

Forbidden replacements are

```text

estimate the four Sonin pieces in (BD.21) separately. (BD.25)
```
