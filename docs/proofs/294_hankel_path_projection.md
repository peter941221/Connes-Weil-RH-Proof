# Proof 294: Hankel block form of the right-path projection

Date: 2026-07-16

Status: exact finite-horizon block formula for Proof 290's right-path
projection after Proof 293's observable compression.  The emission/emission
corner is a Hankel matrix in the path indices.  Its detector commutator has the
same anti-diagonal structure.

This proof does not perform the required signed summation by parts, establish
the uniform stopped-path bound, close Gate 3U, prove the finite-`S` sign, or
prove RH.

## 1. Result

Retain

```text
R_n x=(Kx,K Delta x,...,K Delta^n x,Delta^(n+1)x),

P_n=R_n(R_n*R_n)^(-1)R_n*.                         (BE.1)
```

Proof 290 gives

```text
S_n:=R_n*R_n
 =(I+Delta^(2n+3))(I+Delta)^(-1).                  (BE.2)
```

Define

```text
H_n:=S_n^(-1)
 =(I+Delta)(I+Delta^(2n+3))^(-1).                  (BE.3)
```

Since `0<=Delta<=I`,

```text
I<=H_n<=2I.                                        (BE.4)
```

For emission indices `0<=j,k<=n`, the blocks of `P_n` are

```text
(P_n)_(j,k)
 =K Delta^(j+k)H_nK*,                              (BE.5)

(P_n)_(j,tail)
 =K Delta^jH_nDelta^(n+1),                         (BE.6)

(P_n)_(tail,k)
 =Delta^(n+1)H_nDelta^kK*,                         (BE.7)

(P_n)_(tail,tail)
 =Delta^(n+1)H_nDelta^(n+1).                       (BE.8)
```

The emission corner is Hankel:

```text
(P_n)_(j+1,k)=(P_n)_(j,k+1)                        (BE.9)
```

whenever both sides exist.  The block depends on `j+k`, not on `j-k`.

## 2. Direct derivation

Write the block rows of `R_n` as

```text
R_(n,j)=K Delta^j              for 0<=j<=n,
R_(n,tail)=Delta^(n+1).                         (BE.10)
```

Then

```text
(P_n)_(a,b)=R_(n,a)H_nR_(n,b)*.                    (BE.11)
```

Both `H_n` and every power of `Delta` are bounded Borel functions of the same
self-adjoint contraction, so they commute.  Substitution of `(BE.10)` into
`(BE.11)` proves `(BE.5)--(BE.8)`.  Equation `(BE.9)` is immediate.

No inverse of `Gamma=I-Delta` appears.  The only inverse is `H_n`, whose
uniform bound is `(BE.4)`.

## 3. Observable detector commutator

Proof 293 replaces every emission detector by

```text
W_E=E W E
```

without changing the ordered response.  Put

```text
O_n^E=diag(W_E,...,W_E,W_B).                        (BE.12)
```

The emission/emission blocks of the path commutator are

```text
[O_n^E,P_n]_(j,k)

 =W_E K Delta^(j+k)H_nK*
  -K Delta^(j+k)H_nK*W_E.                           (BE.13)
```

They inherit the Hankel identity

```text
[O_n^E,P_n]_(j+1,k)
 =[O_n^E,P_n]_(j,k+1).                              (BE.14)
```

The mixed blocks are

```text
[O_n^E,P_n]_(j,tail)
 =W_E K Delta^jH_nDelta^(n+1)
  -K Delta^jH_nDelta^(n+1)W_B,                     (BE.15)

[O_n^E,P_n]_(tail,k)
 =W_B Delta^(n+1)H_nDelta^kK*
  -Delta^(n+1)H_nDelta^kK*W_E.                     (BE.16)
```

The tail/tail block is the analogous band commutator.  Every detector
difference in `(BE.13)--(BE.16)` is generated by Proof 293's visible pair

```text
g_W=E W E K-KW_B,
g_W^left=K*E W E-W_BK*.                             (BE.17)
```

For the scalar fixed mode, all blocks vanish exactly.

## 4. What the Hankel structure buys

The path projection is not Burnol's source projection.  Its usable feature is
the exact discrete transport law `(BE.9)`.  Along the emission grid,

```text
       k ->
    +-------------------
  j | M_0 M_1 M_2 M_3
  | | M_1 M_2 M_3 M_4
  v | M_2 M_3 M_4 M_5

M_r=K Delta^rH_nK*.                                 (BE.18)
```

Moving one step down is the same as moving one step right.  The proposed
analytic move is therefore discrete summation by parts along anti-diagonals,
paired with the recurrence generated by `(BE.17)`.  A successful identity
must leave only:

```text
initial emission boundary;
final survivor boundary;
Proof 293's completed outer-return/Sonin physical boundary.              (BE.19)
```

Only after that reduction may compact root support be applied.

## 5. Why blockwise norms are still forbidden

The uniform bound `I<=H_n<=2I` does not control the sum of absolute values of
all blocks.  There are quadratically many emission pairs, and the source
contraction may have spectrum arbitrarily close to one as `S` grows.

The finite certificate records the diagnostic

```text
sum_(all path blocks) norm([O_n^E,P_n]_(a,b))_F,      (BE.20)
```

but does not use it as an estimate.  From the shortest tested horizon to
`n=32`, this positive block cost grows by `1.692x/1.670x` in the two cohorts,
while the full commutator operator norm remains below `0.88/0.85`.  Finite
saturation is not a uniform source theorem; it only demonstrates that the
blockwise absolute-value route discards cancellation.

Forbidden moves are

```text
sum block trace norms before the anti-diagonal cancellation;
bound the result by the number of path blocks;
identify P_n with the fixed Burnol projection;
drop the tail blocks;
split [W,R] into separately normed Sonin subbranches. (BE.21)
```
