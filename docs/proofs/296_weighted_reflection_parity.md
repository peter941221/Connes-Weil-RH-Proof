# Proof 296: Weighted reflection parity and the one-hard-sector reduction

Date: 2026-07-16

Status: exact finite-horizon decomposition of the actual Proof 290 off-range
owner

```text
Z_n* [O_n^E,P_n] R_n.
```

The interior path grid splits into four reflection-parity sectors.  Three
sectors have at least two fixed-survival zeros.  The all-even sector has only
one zero and must remain paired with the path boundary strip and Proof 293's
physical outer-return/Sonin generator.

This proof rejects the stronger claim that reflection makes the entire
weighted path owner double-zero.  It does not prove the uniform source bound,
close Gate 3U, prove the finite-`S` sign, or prove RH.

## 1. Actual weighted owner

Put `N=n+1` and index the path by

```text
I_N={0,1,...,N-1,N},
```

where `N` is the survivor tail.  Proofs 290 and 294 give

```text
R_a=K Delta^a                    for 0<=a<N,
R_N=Delta^N,

F_a=R_a H_n,
Z_a=L_a-F_a,

H_n=h_N(Delta),
h_N(t)=(1+t)/(1+t^(2N+1)).                         (BG.1)
```

For emission indices,

```text
Z_a=K(I-f_(N,a)(Delta)),
f_(N,a)(t)=t^a h_N(t),                             (BG.2)
```

while

```text
Z_N=I-f_(N,N)(Delta).                              (BG.3)
```

Let

```text
C_(a,b)=[O_n^E,P_n]_(a,b).                         (BG.4)
```

The full off-range owner is

```text
T_n^off
 =Z_n*[O_n^E,P_n]R_n
 =sum_(a,b in I_N) Z_a* C_(a,b) R_b.               (BG.5)
```

Equation `(BG.5)` is an operator identity on the band space.  The final root
trace is taken only after the complete sum is assembled.

## 2. Interior and boundary strip

Define the interior and path boundary by

```text
I_N^o={1,...,N-1},
partial I_N={0,N}.                                  (BG.6)
```

Split

```text
T_n^off=T_n^int+T_n^partial,                        (BG.7)

T_n^int
 =sum_(a,b in I_N^o) Z_a*C_(a,b)R_b,               (BG.8)
```

where `T_n^partial` contains every pair with at least one index in
`partial I_N`.

The boundary strip is not a small tail error.  It contains the carrier-change
terms from Proof 295 and remains part of the physical route owner.

## 3. Reflection parity

On the interior, let

```text
rho(a)=N-a.                                         (BG.9)
```

For each `a,b in I_N^o`, define

```text
A_a=Z_a*,             A_a^rho=Z_(rho(a))*,
B_b=R_b,              B_b^rho=R_(rho(b)),
C_(a,b)^rho=C_(rho(a),rho(b)).                      (BG.10)
```

Use even/odd halves

```text
A_a^plus =(A_a+A_a^rho)/2,
A_a^minus=(A_a-A_a^rho)/2,

B_b^plus =(B_b+B_b^rho)/2,
B_b^minus=(B_b-B_b^rho)/2,

C_(a,b)^plus =(C_(a,b)+C_(a,b)^rho)/2,
C_(a,b)^minus=(C_(a,b)-C_(a,b)^rho)/2.             (BG.11)
```

The elementary parity identity is

```text
A C B+A^rho C^rho B^rho

 =2[
    A^plus C^plus B^plus
   +A^plus C^minus B^minus
   +A^minus C^plus B^minus
   +A^minus C^minus B^plus].                        (BG.12)
```

Reflection permutes the whole interior grid.  Summing `(BG.12)` and dividing
by two gives the exact weighted decomposition

```text
T_n^int
 =T_(+++)+T_(+--)+T_(-+-)+T_(--+),                 (BG.13)
```

where, for example,

```text
T_(+--)
 =sum_(a,b in I_N^o)
    A_a^plus C_(a,b)^minus B_b^minus.               (BG.14)
```

This is the actual `Z*CR` weighting, not an unweighted block identity.

## 4. Exact scalar filters

For `1<=a,b<N`, the path halves are

```text
A_a^plus
 =a_(N,a)^plus(Delta)K*,

a_(N,a)^plus(t)
 =1-[f_(N,a)(t)+f_(N,N-a)(t)]/2,                   (BG.15)

A_a^minus
 =a_(N,a)^minus(Delta)K*,

a_(N,a)^minus(t)
 =[f_(N,N-a)(t)-f_(N,a)(t)]/2,                     (BG.16)

B_b^plus
 =K[Delta^b+Delta^(N-b)]/2,                        (BG.17)

B_b^minus
 =K[Delta^b-Delta^(N-b)]/2.                        (BG.18)
```

Let `r=a+b`.  Proof 295 gives

```text
C_(a,b)^plus
 =1/2 [W_E,K(2I-beta_(N,r)(Delta))K*],              (BG.19)

C_(a,b)^minus
 =1/2 [W_E,K delta_(N,r)(Delta)K*],                 (BG.20)

delta_(N,r)=f_(N,r)-f_(N,2N-r).                    (BG.21)
```

Every term in `(BG.15)--(BG.21)` is still generated by the same
`g_W,g_W^left` pair from Proof 293.

## 5. Fixed-mode zero ledger

At `t=1`, the path filters satisfy

```text
a_plus(1)=a_minus(1)=0,
b_minus(1)=0,
delta_(N,r)(1)=0,
beta_(N,r)(1)=beta_(N,r)'(1)=0.                    (BG.22)
```

Their first derivatives are

```text
(a_(N,a)^plus)'(1)=N/2,
(a_(N,a)^minus)'(1)=(N-2a)/2,

(b_(N,b)^minus)'(1)=(2b-N)/2,
delta_(N,r)'(1)=2(r-N).                             (BG.23)
```

The parity ledger is therefore

```text
+----------+--------------------------------------+----------------------+
| sector   | fixed-mode factors                   | minimum zero order   |
+----------+--------------------------------------+----------------------+
| +++      | a_plus; C_plus has physical constant | one                  |
| +--      | a_plus, C_minus, b_minus             | three                |
| -+-      | a_minus, b_minus                     | two                  |
| --+      | a_minus, C_minus                     | two                  |
+----------+--------------------------------------+----------------------+
```

The all-even slope is exactly `N/2`, not zero.  Hence

```text
"reflection makes all weighted bulk double-zero"   is false.           (BG.24)
```

The only single-zero interior sector is

```text
T_(+++).                                             (BG.25)
```

Its constant commutator component is

```text
[W_E,KK*]=g_WK*-Kg_W^left,                          (BG.26)
```

so the hard sector is still a completed physical boundary object.

## 6. Correct analytic split

Proof 296 narrows the hard estimate to

```text
canonical response
  +T_(+++)
  +T_n^partial
  +three soft parity sectors.                       (BG.27)
```

The first three terms must initially remain together: finite models show that
`T_n^partial` carries most of the off-range response, and Proof 264 forbids
discarding the ordered anomaly by a polar cycle.

The three soft sectors have completed double/triple fixed-mode zeros.  This
makes a uniform stopped DOI estimate plausible, but does not prove it.  Their
divided differences must not be normed one block at a time.

The next source theorem is

```text
canonical + T_(+++) + T_n^partial
  -> one complete outer-return/Sonin boundary scalar,

T_(+--)+T_(-+-)+T_(--+)
  -> one beta/delta-filtered stopped remainder,

then compact support,
then one absolute value.                             (BG.28)
```

If `(BG.28)` cannot be proved, the reflection route does not close Gate 3U;
the owner remains Proof 267's relative determinant.

Finite matrices report near-zero traces for each interior parity sector, but
Proof 297 shows that this is not a continuous theorem.  A route-shaped
weighted commutator with trace-class commutator has infinite trace `-i/16`,
