## Lemma 1. Endpoint-Strip Index Set

Statement:

```text
EndpointStripIndexSet(S,I,lambda,J):
  for fixed finite S, compact interval I, cutoff lambda, and graph order J,
  the projection-order defects generated by

    [P,M_S], [P_hat,M_S], [P,M_S^*], [P_hat,M_S^*]

  determine a finite endpoint-strip index set R_(S,I,lambda,J).
```

Proof.

For fixed finite `S`, the Euler multiplier expansions have a fixed coefficient
budget at graph order `J`. The support and Fourier-support windows have fixed
boundaries after `I` and `lambda` are fixed. Commuting a finite-S shift through
one of these projections creates a multiplier supported only near the shifted
boundary.

Thus each endpoint-strip normal-form term can be indexed by:

```text
alpha=(r,s,a,X_0,X_1,b,T_a),
```

where:

```text
r+s <= J,
supp(b) subset E_(lambda,a),
X_0,X_1 are fixed graph-bounded factors,
T_a is a bounded shift/dilation factor.
```

For fixed `S,I,lambda,J`, the family of such terms is finite after grouping
the fixed-S coefficient budget and graph factors into `R_(S,I,lambda,J)`.

## Lemma 2. Cdef Norm Formula

Statement:

```text
CdefNormFormula(S,I,lambda,J):
  Cdef_(S,I,lambda,J)(g) is the trace-norm sum over endpoint-strip normal-form
  terms and the boundary-strip terms created by Q.
```

Definition.

Let `Q R_(S,I,lambda,J)` be the finite family produced by applying the
Connes-Consani `Q` trace-remainder formula to the same endpoint-strip
normal-form terms. Define:

```text
Cdef_(S,I,lambda,J)(g)
  :=
sum_(alpha in R_(S,I,lambda,J))
  || theta(D^r g) X_0 M_b T_a X_1 theta(D^s g)^* ||_1

  +
sum_(beta in Q R_(S,I,lambda,J))
  |BoundaryStripTrace_beta(g)|.
```

Each boundary term in the second sum contains at least one endpoint-strip
factor before boundary evaluation.

Why this is the correct norm:

```text
projection-order defect
  ->
endpoint-strip normal form
  ->
trace-class term or Q-boundary strip term
  ->
one summand of Cdef.
```

No no-strip rank or pole channel is included in `Cdef`; those are handled by
the Battle 1 ledger.

## Lemma 3. Endpoint-Strip Trace-Norm Bound

Statement:

```text
EndpointStripTraceNormBound(S,I,lambda,J):
  every normal-form summand in Cdef is trace-class and satisfies the finite
  strip Hilbert-Schmidt bound.
```

Proof.

For one summand:

```text
theta(D^r g) X_0 M_b T_a X_1 theta(D^s g)^*,
```

factor through the endpoint strip:

```text
L2(R)
  --B_s-->
L2(E_(lambda,a))
  --A_r-->
L2(R).
```

The Hilbert-Schmidt estimates give:

```text
||A_r||_2^2
  <=
|E_(lambda,a)| ||K_(D^r g,X_0)||_2^2,

||B_s||_2^2
  <=
||b||_2^2 ||K_(D^s g,X_1)||_2^2.
```

Therefore:

```text
||A_r B_s||_1
  <=
||A_r||_2 ||B_s||_2.
```

Summing the fixed-S coefficient budget gives a constant depending on
`S,I,J` and fixed seminorms of `g`, not on a growing place set.

## Lemma 4. Q Endpoint-Strip Stability

Statement:

```text
QEndpointStripStability(S,I,lambda,J):
  applying the Connes-Consani Q trace-remainder extraction to an endpoint-strip
  projection defect keeps an endpoint-strip factor before every boundary
  evaluation.
```

Proof.

The `Q` formula contributes a bounded number of scaling derivatives and
boundary evaluations. Scaling derivatives commute with fixed dilations in
logarithmic coordinates and raise only the graph order.

The middle factor:

```text
M_b T_a
```

remains present in every term generated from a projection defect. Hence bulk
terms still factor through `L2(E_(lambda,a))`.

For boundary terms, the evaluation functional is composed with a strip
factor:

```text
evaluation o theta(D^r g) X_0 M_b T_a.
```

The trace theorem on a finite strip gives:

```text
|evaluation(A_r v)|
  <=
C_(S,I,J)(g) ||v||_(L2(E_(lambda,a))).
```

Thus `Q` cannot turn a projection-defect term into a no-strip rank or pole
jet. The only price is a larger fixed graph order, written `J'`.

## Lemma 5. Graph Comparison

Statement:

```text
CdefGraphComparison(S,I,lambda,J,J'):
  Cdef_(S,I,lambda,J)(g)
    <=
  C'_(S,I,J)(g) Cdef_graph_(S,I,lambda,J')(g).
```

Proof.

`Cdef_graph` is the graph/prolate control quantity used earlier in the route.
It controls the same endpoint-strip Hilbert-Schmidt factors and the fixed
finite collection of boundary-strip evaluations after the `Q` graph-order
increase.

Lemmas 3 and 4 show that every trace-norm summand in `Cdef` is controlled by
one of those graph/prolate seminorms. Since `R_(S,I,lambda,J)` and
`Q R_(S,I,lambda,J)` are finite for fixed parameters, summing the inequalities
gives:

```text
Cdef_(S,I,lambda,J)(g)
  <=
C'_(S,I,J)(g) Cdef_graph_(S,I,lambda,J')(g).
```

The constant may depend on the fixed finite set `S`, the interval `I`, the
graph orders, and fixed seminorms of `g`. It must not depend on a place set
growing with `lambda`.

## Lemma 6. Fixed-Test Cdef Exhaustion

Statement:

```text
FixedTestCdefExhaustion(S_A,I,g,J'):
  Cdef_graph_(S_A,I,lambda,J')(g) -> 0
  as lambda -> infinity,
```

with:

```text
g, I, S_A, J' fixed.
```

Proof.

The order of choices is:

```text
1. choose the compactly supported test g;
2. choose I containing supp(g);
3. choose A with supp(g^* * g) subset exp([-A,A]);
4. set S_A={infinity} union {p : log p <= A};
5. fix graph order J';
6. send lambda -> infinity.
```

With `g` and `S_A` fixed, the finite-S Euler multipliers and graph constants
are fixed. The cutoff `lambda` only exhausts the support and Fourier-support
windows. The fixed-test graph exhaustion package gives:

```text
Cdef_graph_(S_A,I,lambda,J')(g) -> 0.
```

By Lemma 5:

```text
Cdef_(S_A,I,lambda,J)(g)
  <=
C'_(S_A,I,J)(g) Cdef_graph_(S_A,I,lambda,J')(g).
```

The constant is fixed in this limit, so:

```text
Cdef_(S_A,I,lambda,J)(g) -> 0.
```

## Consequence For The Triple-Killed Inequality

Assume Battles 1 and 2 have produced:

```text
QW_lambda(g,g)
  >=
-C_(S_A,I,J)(g) Cdef_(S_A,I,lambda,J)(g)
```

for the fixed test `g` and the fixed finite set `S_A`. Battle 3 gives:

```text
Cdef_(S_A,I,lambda,J)(g) -> 0.
```

Therefore:

```text
liminf_(lambda -> infinity) QW_lambda(g,g) >= 0.
```

The convergence from `QW_lambda(g,g)` to the full `QW(g,g)` is then a separate
source-read-off/exhaustion statement:

```text
finite-prime part:
  support of g^* * g makes the prime-power sum stable for large lambda;

pole part:
  pole evaluations are independent of lambda;

archimedean part:
  the restricted CCM cutoff exhausts the fixed half-density test.
```

Battle 3 supplies the defect-limit step in this chain.
