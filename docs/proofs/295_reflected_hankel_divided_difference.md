# Proof 295: Reflected Hankel divided-difference pair

Date: 2026-07-16

Status: exact finite-horizon refinement of Proof 294.  Every path-projection
commutator block is generated by Proof 293's single visible boundary pair
through an operator divided difference.  Reflection across the finite path
pairs interior Hankel blocks into a positive contraction which has a double
zero at the fixed survival mode.  The `0/tail` pairs require explicit
`g_W,g_W^left` boundary corrections.

This proof does not yet assemble the reflected pairs inside the complete
Proof 293 scalar, prove the uniform compact-support estimate, close Gate 3U,
prove the finite-`S` sign, or prove RH.

## 1. Result

Put

```text
N=n+1,
h_N(t)=(1+t)/(1+t^(2N+1)),
f_(N,r)(t)=t^r h_N(t),       0<=r<=2N.              (BF.1)
```

Proof 294's inverse Gram is

```text
H_n=h_N(Delta).                                      (BF.2)
```

Define the reflected fixed-mode defect

```text
beta_(N,r)(t)
 =2-f_(N,r)(t)-f_(N,2N-r)(t).                       (BF.3)
```

For every `0<=t<=1`,

```text
0<=beta_(N,r)(t)<=2,
beta_(N,r)(1)=0,
beta_(N,r)'(1)=0.                                    (BF.4)
```

Thus

```text
B_(N,r):=beta_(N,r)(Delta)
```

is a positive contraction up to the factor `2`, and the reflected pair has
two fixed-mode zeros before any norm is taken.

## 2. Why the reflected scalar is positive

For fixed `t in [0,1]`, the function

```text
r -> t^r+t^(2N-r)
```

is convex and symmetric on `[0,2N]`.  Its maximum occurs at an endpoint, so

```text
t^r+t^(2N-r)<=1+t^(2N).                              (BF.5)
```

Also

```text
2(1+t^(2N+1))-(1+t)(1+t^(2N))
 =(1-t)(1-t^(2N))>=0.                                (BF.6)
```

Equations `(BF.5)--(BF.6)` prove the lower bound in `(BF.4)`; the upper bound
follows because both `f` terms are nonnegative.

At the fixed mode,

```text
h_N(1)=1,
h_N'(1)=-N,
f_(N,r)'(1)=r-N.                                     (BF.7)
```

The reflected exponent has derivative `N-r`, proving the double zero in
`(BF.4)`.  Equivalently,

```text
beta_(N,r)(t)=(1-t)^2 q_(N,r)(t)                    (BF.8)
```

for a continuous nonnegative `q_(N,r)`.  The size of `q` can be `O(N^2)`;
only the bounded completed product `beta`, not `q` by itself, is a candidate
uniform resource.

## 3. Divided-difference generation

Let

```text
W_E=E W E,
W_B=iota_B*W iota_B,

g_W=W_EK-KW_B,
g_W^left=K*W_E-W_BK*,

C_W=[W_B,Delta]=g_W^left K-K*g_W.                   (BF.9)
```

For a scalar function `a` on the spectrum of `Delta`, define its divided
difference by

```text
a^[1](lambda,mu)
 =(a(mu)-a(lambda))/(mu-lambda),                    (BF.10)
```

with diagonal value `a'(lambda)`.  Let

```text
DOI_a^Delta(X)
 =integral integral
    a^[1](lambda,mu) dE_Delta(lambda) X dE_Delta(mu).
                                                               (BF.11)
```

For the rational functions in `(BF.1)`, the ordinary commutator identity is

```text
[W_B,a(Delta)]=DOI_a^Delta(C_W).                    (BF.12)
```

Consequently every emission moment

```text
M_(N,r)=K f_(N,r)(Delta)K*                          (BF.13)
```

has the exact detector commutator

```text
[W_E,M_(N,r)]

 =g_W f_(N,r)(Delta)K*
  -K f_(N,r)(Delta)g_W^left
  +K DOI_(f_(N,r))^Delta(C_W)K*.                    (BF.14)
```

The mixed and tail blocks are generated by the same pair:

```text
W_E K f-K f W_B
 =g_W f+K DOI_f^Delta(C_W),                         (BF.15)

W_B f K*-f K*W_E
 =DOI_f^Delta(C_W)K*-f g_W^left,                   (BF.16)

[W_B,f]=DOI_f^Delta(C_W).                           (BF.17)
```

No new detector owner appears as the horizon or block index changes.

## 4. Interior reflected blocks

Index the path by

```text
0,1,...,N-1,N,
```

where `N` is the survivor tail.  For `1<=j,k<=N-1`, both reflected indices
`N-j,N-k` are emission indices.  Proof 294 and `(BF.3)` give

```text
P_(j,k)+P_(N-j,N-k)
 =K(2I-B_(N,j+k))K*.                                (BF.18)
```

For

```text
C_(a,b)=[O_n^E,P_n]_(a,b),                          (BF.19)
```

the corresponding commutator identity is

```text
C_(j,k)+C_(N-j,N-k)
 =[W_E,K(2I-B_(N,j+k))K*].                          (BF.20)
```

The common constant part is already physical:

```text
[W_E,KK*]=g_WK*-Kg_W^left.                          (BF.21)
```

The `B_(N,r)` remainder is also generated by `(BF.14)`, with `f` replaced by
`beta`.  Thus reflection preserves the single-generator ownership.

## 5. Completed 0/tail reflection

The path endpoints have different carriers: emission `0` lies in the global
`E` space, while tail `N` lies in the band space.  Multiplication by `K` and
`K*` is therefore part of the exact reflection.

For `1<=k<=N-1`,

```text
P_(0,k)+K P_(N,N-k)
 =K(2I-B_(N,k))K*,                                  (BF.22)

C_(0,k)+K C_(N,N-k)+g_W P_(N,N-k)
 =[W_E,K(2I-B_(N,k))K*].                            (BF.23)
```

For `1<=j<=N-1`,

```text
P_(j,0)+P_(N-j,N)K*
 =K(2I-B_(N,j))K*,                                  (BF.24)

C_(j,0)+C_(N-j,N)K*-P_(N-j,N)g_W^left
 =[W_E,K(2I-B_(N,j))K*].                            (BF.25)
```

At the corner,

```text
P_(0,0)+K P_(N,N)K*
 =K(2I-B_(N,0))K*,                                  (BF.26)

C_(0,0)+K C_(N,N)K*
 +g_W P_(N,N)K*-K P_(N,N)g_W^left
 =[W_E,K(2I-B_(N,0))K*].                            (BF.27)
```

The extra terms in `(BF.23)`, `(BF.25)`, and `(BF.27)` are not optional
errors.  They are the exact carrier-change boundary corrections.  A
near-survival finite guard makes the naive formulas without them fail at
order `10^-2`, far above algebraic roundoff.

## 6. What remains open

summation.  The next theorem must insert `(BF.20)--(BF.27)` into

```text
F_n*E_n^vis+L_n*(I-P_n)E_n^vis                      (BF.28)
```

with the actual left/right path weights.  It must show whether the complete
anti-diagonal pairing leaves only

```text
the beta-filtered double-zero bulk;
the explicit g_W,g_W^left endpoint corrections;
the completed outer-return/Sonin physical boundary. (BF.29)
```

No absolute value may be placed between these terms.  If the actual path
weights do not pair the reflected blocks, the reflection lane is rejected;
the scalar remains on Proof 267's relative determinant line.
