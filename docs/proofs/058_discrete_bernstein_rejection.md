# Discrete Bernstein Route Rejection

Date: 2026-07-12

Status: the square-node sequence generated by the exact coupled Weil source
fails the first nontrivial finite-difference conditions required by both a
discrete Bernstein representation and a positive Stieltjes/Bernstein
representation on the squared nodes. The failure is Arb-certified at the
first three positive frequencies for three prime cutoffs. Finite Pick
interpolation after a positive Loewner certificate is circular and supplies no
uniform lower owner.

## 1. Candidate Positive-Measure Structures

Write

```text
a_n = Phi_c(n^2), n>=1,
```

for the Loewner generator identified in
`056_loewner_bernstein_rejection.md`.

Two lattice-specific escapes remained after continuous operator monotonicity
failed.

First, a discrete Bernstein representation in the integer parameter has the
form, up to harmless affine terms,

```text
a_n = alpha + beta*n + integral_[0,1) (1-r^n) d mu(r), mu>=0.
```

Its increments are

```text
a_(n+1)-a_n = beta + integral r^n(1-r) d mu(r),
```

and must be nonincreasing.

Second, a Bernstein/Stieltjes representation only on the squared nodes would
be inherited from

```text
Phi(x) = alpha + beta*x + integral_(0,infinity) x/(x+t) d nu(t), nu>=0.
```

Such a function is concave. Therefore its adjacent square-node secant slopes

```text
s_n = (a_(n+1)-a_n)/((n+1)^2-n^2)
```

must be nonincreasing. These are necessary conditions independent of RH or
finite matrix diagonalization.

## 2. Arb-Certified Failure

The probe evaluates `Phi_c(n^2)` using the same official closed forms already
checked against the finite Loewner matrix. At `c=13`:

```text
a_1 = 0.0457203442394000540...
a_2 = 0.187963689777148372...
a_3 = 0.446258031000765792...

(a_3-a_2)-(a_2-a_1)
  = +0.116050995685869103...

(a_3-a_2)/5-(a_2-a_1)/3
  = +0.00424441973214071156...
```

Both required monotonicities go in the wrong direction. The independent
cutoffs agree:

```text
+-----+----------------------+----------------------+
| c   | raw increment growth | square-secant growth |
+-----+----------------------+----------------------+
| 13  | +0.116050995686...   | +0.00424441973214... |
| 29  | +0.0897200191169...  | +0.00258041827527... |
| 100 | +0.0554538429552...  | +0.000847128435811... |
+-----+----------------------+----------------------+
```

Every value is an Arb interval strictly above zero. For example, at `c=100`
the radius of the smaller square-secant growth is below `1e-29`, while its
midpoint is about `8.47e-4`.

Failure at this first difference level excludes every higher complete-
monotonicity hierarchy built on the same positive measures; checking higher
orders cannot repair it.

## 3. Why Finite Pick Interpolation Is Not A Survivor

A finite set of node values may admit many monotone scalar interpolants. That
fact does not imply positivity of its Loewner matrix. If diagonal derivative
data are included, feasibility of a Pick/Nevanlinna interpolation is encoded
by the relevant Pick/Loewner positivity condition itself.

Consequently, for each already certified positive `Q_(c,N)`, one may recover
an `N`-dependent rational Pick function or Gram factor. But constructing it by
LDL, Cholesky, or a solved Pick matrix consumes the finite sign being proved.
No coherent arithmetic measure independent of `N` results, and positivity for
all `N` remains the constrained Weil gate.

This leaves no lattice Bernstein/Stieltjes shortcut:

```text
explicit common positive measure: rejected by finite differences
N-dependent post-PSD measure: circular certificate
scalar monotone interpolation: too weak to prove matrix positivity
```
