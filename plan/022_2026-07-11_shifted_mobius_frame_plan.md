# Plan 022: Moment-Zero Shifted-Mobius Frame

Date: 2026-07-11

Status: rejected as an executable RH route. The fixed frame and anchored
convolution survive numerical angle tests, but the angle is generated by the
full high-dimensional optimal shell geometry. The explicit multiscale
convolution dictionary itself has nonzero limiting distance, so proving the
angle reintroduces the Schur/inverse-Gram bottom that Plan 022 was meant to
remove.

The second rejection round selects one fixed anchored convolution direction as
the primary proof object. The eight-direction frame remains a positive-square
fallback, not the first theorem target.

This plan was derived inside the project from finite sequence identities and
GPU experiments. It does not use an external theorem as its mathematical
producer or claim historical novelty.

## 1. Scope And Non-Goals

Work in the discrete weighted Hilbert space

```text
H = {h : sum_(n>=1) |h(n)|^2/(n*(n+1)) < infinity},
gamma(n)=1,
gamma_k(n)={n/k},
V_N=span(gamma_2,...,gamma_N),
r_N=gamma-P_N gamma,
d_N^2=||r_N||^2.
```

The plan seeks a fixed finite family of explicit directions in
`span(gamma_(N+1),...,gamma_(2N))` whose conservative aggregate capture is at
least `c/log N` of `d_N^2`.

Non-goals:

```text
do not fit coefficients separately at each N
do not read r_N, P_N, or G_N^-1 when constructing the directions
do not use projected Schur energy in the proposed lower certificate
do not infer a uniform theorem from finite GPU data
do not create a Lean RH route before the correlation frame bound is proved
```

## 3. Fixed Frame

Fix `D=8`. For `1<=d<=D` and `N<k<=2N`, define

```text
q_(N,d)(k) = 1_(d divides k) * mu(k/d),
H_N        = sum_(N<j<=2N) 1/j,
c_(N,d)    = (sum_(N<j<=2N) q_(N,d)(j)/j) / H_N,
a_(N,d)(k) = q_(N,d)(k)-c_(N,d),
b_(N,d)    = sum_(N<k<=2N) a_(N,d)(k) gamma_k.
```

The harmonic correction gives

```text
sum_(N<k<=2N) a_(N,d)(k)/k = 0.
```

For every `n<=N`, all block indices satisfy `k>n`, so `gamma_k(n)=n/k` and

```text
b_(N,d)(n)=n*sum_k a_(N,d)(k)/k=0.              (F0)
```

Thus every frame vector has an exact coordinate support gap before the dyadic
block. This is stronger than small old-space overlap and requires no optimal
projection to define.

## 4. Conservative Capture Bridge

Let `B_N=span(b_(N,1),...,b_(N,D))`. Define the unprojected frame Gram and
correlation vector

```text
U_N(d,e)=<b_(N,d),b_(N,e)>,
z_N(d)=<r_N,b_(N,d)>.
```

For any `b in B_N`, put `w=(I-P_N)b`. Since `r_N` is orthogonal to `V_N`,

```text
<r_N,w>=<r_N,b>,
||w||<=||b||.
```

Therefore the actual decrement available inside `V_(2N)` is at least

```text
sup_(b in B_N) |<r_N,b>|^2/||b||^2
  = z_N^T U_N^-1 z_N.                              (F1)
```

This deliberately uses the larger unprojected denominator. It removes the old
Schur complement from the certificate. If there are `c>0` and `N_0` such that

```text
z_N^T U_N^-1 z_N >= (c/log N)*d_N^2               (F2)
```

for every dyadic `N>=N_0`, the already-checked dyadic product argument proves
`d_N->0` and hence RH.

## 4A. Primary Fixed Convolution Direction

Define the fixed kernel

```text
theta(d) = -mu(d)/d for 1<=d<=8, and 0 otherwise,
Q(k)     = sum_(d|k) theta(d)*mu(k/d).
```

For squarefree `k`, this simplifies exactly to

```text
Q(k) = -mu(k) * sum_(d|k, d<=8) 1/d.                 (F3)
```

Let

```text
S_N = sum_(N<k<=2N) Q(k)/k,
h_N = sum_(N<k<=2N) Q(k) gamma_k - N*S_N*gamma_N.   (F4)
```

The old anchor does not affect correlation with `r_N`, because
`r_N` is orthogonal to `gamma_N`. For every `n<N`, both terms in (F4) equal
`n*S_N`, so

```text
h_N(n)=0.                                            (F5)
```

The discrete gradient identity is

```text
Delta h_N(n)
  = -sum_(N<k<=2N, k|n) Q(k) + N*S_N*1_(N|n).       (F6)
```

The global coefficient convolution has the exact collapse

```text
1*Q = (1*mu)*theta = theta.                          (F7)
```

Thus all divisor complexity in the untruncated convolution reduces to the
fixed support `1<=d<=8`. The truncation boundary and the old anchor are the only
remaining energy terms. This is the preferred entrypoint for an exact bound.

The sufficient fixed-direction theorem is

```text
|<r_N,h_N>|^2 / ||h_N||^2 >= (c/log N)*d_N^2.       (F8)
```

Unlike the original 020 direction, `h_N` has an exact initial coordinate gap,
a fixed coefficient formula, an unprojected conservative denominator, and the
finite divisor collapse (F7).

## 6. Exact Mathematical Bottom

The next work is not a larger GPU run. It is the pair of finite estimates

```text
frame energy upper bound:
  lambda_max(U_N) <= explicit C_U(N)

aggregate correlation lower bound:
  sum_(d=1)^8 |<r_N,b_(N,d)>|^2
    >= explicit C_z(N)*d_N^2/log N
```

with `C_z(N)/C_U(N)` bounded below by a positive constant. Every Gram entry has
the existing finite periodic digamma formula. The new numerator must be
expanded as a positive aggregate of eight squared correlations and reduced to
finite divisor sums. A proof that merely rewrites `r_N` through `G_N^-1` does
not pass.

The initially proposed sharper pair was:

```text
anchored convolution energy:
  ||h_N||^2 <= C_h

anchored convolution correlation:
  |<r_N,h_N>|^2 >= (c_h/log N)*d_N^2.               (F9)
```

Use (F6)--(F7) before expanding a Gram matrix. If the energy proof requires a
general moment-zero Bessel inequality, reject that proof: its generalized
largest eigenvalue grows linearly in `N` numerically. The estimate must use the
specific double-Mobius convolution.

The second round rejects proving these two estimates independently. Let

```text
P_Q(x)=sum_(k<=x) Q(k),
M(x)=sum_(k<=x) mu(k).
```

Since `Q=theta*mu`, finite convolution gives

```text
P_Q(x)=sum_(d=1)^8 theta(d) M(floor(x/d)).            (F10)
```

Because `theta(1)=-1`, this can be inverted recursively:

```text
M(x)=-P_Q(x)+sum_(d=2)^8 theta(d)M(floor(x/d)).      (F11)
```

Moreover

```text
sum_(d=2)^8 |theta(d)|/sqrt(d) < 0.758 < 1.          (F12)
```

Thus a pointwise square-root bound for `P_Q` would already give a square-root
Mertens bound by induction. Do not use such a bound as a supposedly lower
energy producer.

The active target returns to the coupled inequality (F8). On `N<n<2N`,

```text
h_N(n)=N*S_N-(P_Q(n)-P_Q(N)).                         (F13)
```

The same centered Mertens bridge must remain in both `<r_N,h_N>` and
`||h_N||`. The new bottom is a uniform angle theorem

```text
|<r_N,h_N>|^2
  >= (c/log N) * ||r_N||^2 * ||h_N||^2.              (F14)
```

Unlike Plan 020, the second vector in this angle is fixed, has an exact initial
gap, and has no Schur inverse. But separating its magnitude from its angle can
destroy the observed mechanism and is forbidden.

The exact first-block form is available without a Gram inverse. For
`N<n<2N`, the block sawtooths have not completed more than one period, so

```text
h_N(n) = N*S_N - sum_(N<k<=n) Q(k).                 (F15)
```

At the endpoint,

```text
h_N(2N)=2N*S_N-sum_(N<k<=2N) Q(k).                 (F16)
```

Expand the first term in (F17) using (F15), `Q=theta*mu`, and the eight fixed
values of `theta`. The proof must exhibit a positive finite-divisor main term.
A smooth scaling limit for `r_N` is not available: normalized residual block
profiles have unstable and often negative adjacent-scale correlations.

## 7. Rejection Guards

Reject Plan 022 if any of the following occurs:

```text
the fixed D=8 certificate has a verified zero or tends to zero
the aggregate lower bound requires Mertens square-root cancellation
the only proof reintroduces the old Schur complement or G_N^-1
the eight correlations require per-scale fitted signs or coefficients
the finite periodic expansion has cancellation of the same size as its main term
the fixed correlation is only recoverable by expanding r_N through G_N^-1
the proof replaces r_N by a smooth scale-limit profile
the proof assumes pointwise positivity of r_N(n)h_N(n)
the proof separates a square-root Mertens bound for P_Q as an energy input
```

Accepted feasibility requires a named positive finite-divisor main term and a
strictly smaller unconditional error. Accepted RH route additionally requires
the exact criterion bridge and a no-argument Lean theorem with no project root.
